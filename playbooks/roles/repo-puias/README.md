# Ansible Role: PUIAS Repository

Installs the PUIAS repository for RHEL/CentOS 6.x.

## Requirements

None.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

    - hosts: servers
      roles:
        - { role: puias }
